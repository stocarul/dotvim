dotvim

My .vim directory!

Clone it into .vim folder:

git clone --recursive https://github.com/stocarul/dotvim.git .vim

Add to ~/.vimrc:

source ~/.vim/vimrc

Install bundles:

vi ~/.vim/vimrc
:BundleInstall

Install additional resources:

sudo apt-get install ack-grep
